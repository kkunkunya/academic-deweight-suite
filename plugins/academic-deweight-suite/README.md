# academic-deweight-suite

Academic deweighting toolkit for papers, theses, grants, and technical reports.

The suite replaces the previous monolithic `previous monolithic academic deweight skill` with six gate-oriented skills:

| Skill | Purpose |
|---|---|
| `academic-deweight-suite:orchestrator` | Main orchestrator and route selector |
| `academic-deweight-suite:deweight-intake-anchor` | Integrity Snapshot + Semantic Anchor |
| `academic-deweight-suite:deweight-coverage-scan` | Full coverage scan with mandatory subagent isolation for long documents |
| `academic-deweight-suite:deweight-human-anchor-bind` | Bind suspect paragraphs to human-writing practice cards |
| `academic-deweight-suite:deweight-anchor-rewrite` | Rewrite only from a bound prescription |
| `academic-deweight-suite:deweight-verify-gate` | Residual AI, new-pattern, and semantic drift verification |

Runtime truth sources live under `_shared/references/`. Local research copies and full-text corpora remain user-provided source material, not public runtime truth.
