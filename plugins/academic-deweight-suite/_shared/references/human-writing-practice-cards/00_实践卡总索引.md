# 通用真人写法实践卡库总索引

## 0. 用途

本库服务于 `academic-deweight-suite`：当文本被识别出 A1-A39 等 AI 痕迹后，不让模型从零“凭感觉”改写，而是先调用相同写作功能下的真人论文摘录，学习其结构动作，再在原文事实约束内重写。

核心调用顺序：

```text
识别 AI 模式 -> 判断段落功能 -> 查 00_A1-A39 或 00_写作功能矩阵 -> 打开具体论文实践卡卡 -> 按结构骨架改写
```

## 5. 使用边界

- 本库保存的是“结构动作”和“证据组织方式”，不是句式模板库。
- 原文摘录来自 MinerU `full.md`，有 OCR/公式/断词噪音；改写时不得模仿抽取错误。
- 英文摘录保留原文短片段；中文结构解释用于指导改写。
- 医学、管理、生物等领域的证据强度边界不可跨学科乱套。
- A9、A10、A23、A39 等模式多是“反向实践卡”：重点学正式论文如何避免聊天腔、视觉噪音、卡片式标题和换风格陷阱。
